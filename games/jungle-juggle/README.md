# Jungle Juggle: Vine Swap

A hyper-casual HTML5 game built with Canvas 2D. Run along a narrow beam, swap between two overhead vines, dodge hazards, and collect coins.

## Play Now

**[https://balinti.github.io/jungle-juggle/](https://balinti.github.io/jungle-juggle/)**

## How to Play

- **Tap / Click / Space / Enter** to swap vines (left/right)
- Stay on the beam - falling off the edge or into a gap means game over
- Collect coins for points - valuable coins (x3) appear on the risky side
- Watch for monkey mischief - monkeys can block a vine temporarily
- Near-miss bonus for skimming the beam edge
- Combo multiplier for consecutive pickups

## Features

- Single HTML file, no dependencies
- Canvas 2D rendering with high-DPI support
- Increasing difficulty (speed, gaps, hazards, narrower beam)
- Particle effects, screen shake, HSL color cycling
- WebAudio sound effects with mute toggle
- localStorage high score persistence
- Pause on tab switch
